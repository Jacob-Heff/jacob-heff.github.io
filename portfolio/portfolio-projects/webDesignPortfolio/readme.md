# Web Development and Design Portfolio

## Description

The folders labled challenge # are based on the following list gathered from the [10 HTML and CSS Code Challenges for Beginners](http://a.comhttps://www.codecademy.com/resources/blog/html-and-css-code-challenges-for-beginners/).
- Challenge 1 - Create a tribute page
- Challenge 2 - Create a Wikipedia page
- Challenge 3 - Create an HTML and CSS-only search results page
- Challenge 4 - Create a survey form
- Challenge 5 - Create a parallax website
- Challenge 6 - Create a product landing page
- Challenge 7 - Create a restaurant website
- Challenge 8 - Create a chessboard
- Challenge 9 - Create an event or conference website
- Challenge 10 -  Create a portfolio website
