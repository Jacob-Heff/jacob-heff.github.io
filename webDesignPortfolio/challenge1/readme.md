# Challenge 1 - Tribute Page

## Description
Choose a historical figure who's meaningful to you and create a webpage dedicated to them. 

The figure I chose was Alan Turing. Turing's importance to mathematics, and more specifically computer science cannot ever be understated.

### **Features**
- A heading with using Turing's name
- An image of him
- A short summary of his importance
- A timeline of the person's life in the form of a panel gallery

## Visuals
Depending on what you are making, it can be a good idea to include screenshots or even a video (you'll frequently see GIFs rather than actual videos). Tools like ttygif can help, but check out Asciinema for a more sophisticated method.

## Roadmap
This project was developed using a StartBootstrap template, there are a lot of details in the code that are no longer used with this design. One development I have is to remove the excess code from the CSS and JS files.
